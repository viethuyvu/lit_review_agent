# Research Navigator – Agentic Literature Discovery Tool

## Setup

1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file with:
    ANTHROPIC_API_KEY=your_anthropic_key
    SEMANTIC_SCHOLAR_API_KEY=your_optional_key
    HF_TOKEN = your_hugging_face_key_optional

## Run

```bash
python main.py
